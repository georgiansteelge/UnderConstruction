# site3 – Georgian Steel (HTML + CSS + JS)

სტატიკური საიტი მხოლოდ HTML, CSS და JavaScript-ით. არანაირი ფრეიმვორკი.

## სტრუქტურა

- `index.html` – მთავარი გვერდი (ყველა სექცია)
- `css/style.css` – სტილები (ბრენდის ფერები: #F47C20, #2D3E4F)
- `js/main.js` – Hero სლაიდერი, countdown, მობილური მენიუ, testimonials, scroll-to-top, newsletter
- `img/` – სურათები (სურვილისამებრ)

## გაშვება

გახსენი `index.html` ბრაუზერში ან გამოიყენე ლოკალური სერვერი (მაგ. Live Server).
